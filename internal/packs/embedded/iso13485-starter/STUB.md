# ISO 13485:2016 — coming soon in v0.2

This pack is reserved at v1 launch but not yet populated with working
content. ISO 13485 builds on ISO 9001 (this pack will declare a
dependency on the iso9001 pack).

The OpenDelve v0.2 roadmap will add:
- Design and development file skeletons
- Complaint handling SOP
- CAPA (corrective and preventive action) SOP
- Lot or batch release SOP with traceability
- Software change review for SaMD
- Mappings to MDR / FDA QMSR

Track v0.2 progress at https://github.com/lloydarmbrust/opendelve.

If you need ISO 13485 content sooner, see [GSTT-CSC/QMS-Template](https://github.com/GSTT-CSC/QMS-Template)
(Apache-2.0) for templates we'll be adapting.
